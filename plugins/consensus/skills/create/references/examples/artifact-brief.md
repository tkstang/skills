# Artifact Brief

Create a concise release announcement for a developer-tooling feature.

## Audience

Engineers and technical team leads who already use the plugin locally.

## Core Points

- The feature adds consensus creation from a brief.
- The default mode asks two peers to draft independently, then synthesizes a final artifact.
- The output should mention that the run includes a deliberation log and resolution metadata.

## Tone

Practical, direct, and suitable for a changelog-adjacent announcement.
